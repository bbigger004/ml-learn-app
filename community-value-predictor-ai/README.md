# 社区价值预测 AI (Community Value Predictor AI)

这是一个基于 React 和 ml5.js 构建的前端机器学习应用。它可以读取 CSV 数据，训练神经网络回归模型，并预测某个小区的 'y' 值。

## 功能特性

1.  **本地数据训练**：上传 CSV 文件进行本地浏览器端训练。
2.  **可视化展示**：
    *   实时展示训练过程中的损失（Loss）曲线。
    *   训练完成后展示“预测值 vs 真实值”的散点图。
3.  **参数微调**：可调整训练轮数 (Epochs)、批次大小 (Batch Size)、学习率 (Learning Rate) 等超参数。
4.  **特征选择**：手动勾选参与训练的输入列。
5.  **模型持久化**：支持保存（下载）训练好的模型权重，以及加载已有模型。

## 如何本地部署与使用

本项目已配置为使用 Vite 进行构建，所有依赖均为本地 npm 依赖。

### 1. 安装依赖

在项目根目录下打开终端，运行：

```bash
npm install
```

### 2. 启动开发服务器

运行以下命令启动本地开发环境：

```bash
npm run dev
```

打开浏览器访问控制台输出的地址（通常是 `http://localhost:5173`）。

### 3. 数据格式说明

请准备一个 CSV 文件，其中必须包含：
*   若干特征列（数值型）
*   一个名为 `y` 的目标列（数值型，我们预测的目标）

**示例 CSV 结构：**

```csv
绿化率,容积率,建筑年代,y
0.35,2.5,2010,50000
0.40,1.8,2015,60000
...
```

### 4. 使用流程

1.  点击左侧栏的 **"上传 CSV 数据"**。
2.  在 **"输入特征"** 列表中勾选需要作为模型输入的列。
3.  调整 **"模型超参数"** (如需要)。
4.  点击 **"开始训练"**。
5.  等待训练完成后，查看右侧的评估图表。
6.  点击 **"保存模型"** 将模型文件下载到本地。
7.  下次使用时，可点击 **"加载模型"** 复用之前的训练结果。
